# NRCS Robotics Society Management System

A professional Object-Oriented C++ based management system designed to simulate and manage the structure of a university robotics society.

This project represents the internal management hierarchy of NRCS (Robotics Society - Riphah International University), including management head, department heads, members, and event management.

------------------------------------------------------------

## 📌 Project Overview

The system models a real-world university robotics society structure:

• 1 Management Head  
• 7 Core Team Members  
• 3 Departments:
    - Tech Department
    - Media Department
    - Documentation Department  
• Department Heads with defined duties  
• Members under each department  
• Event & Seminar Management  

The program demonstrates structured Object-Oriented Programming concepts in C++.

------------------------------------------------------------

## 🏗 System Architecture

The project is built using:

• Classes and Objects  
• Encapsulation  
• Vectors (STL)  
• Modular Design  
• Clean Console Output Formatting  

Main Classes:

1. Member  
   - Stores name and position  

2. Department  
   - Stores department name  
   - Head name  
   - Head duties  
   - Department members  

3. Event  
   - Event name  
   - Date  
   - Type (Seminar / Workshop / Competition)  

4. Society  
   - Society name  
   - Management head  
   - Core team  
   - Departments  
   - Events
